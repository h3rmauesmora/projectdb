# Проект базы данных для школы
## Описание
** Это проект базы данных проката кассет с фильмами, написанный на SQLite**
## Структуры базы данных
**Таблица users (пользователи)**
| Поле | Тип     | Описание                   |
|------|---------|----------------------------|
| id   | INTEGER | Идентификатор пользователя |
| name | TEXT    | Имя пользователя           |
| age  | INTEGER | Возраст пользователя       |
**Таблица tapes (кассеты)**
| Поле     | Тип     | Описание                   |
|----------|---------|----------------------------|
| id       | INTEGER | Идентификатор кассеты      |
| title    | TEXT    | Название фильма на кассете |
| year     | INTEGER | Год премьеры фильма        |
| genre    | TEXT    | Жанр фильма                |
| director | TEXT    | Режиссер фильма            |
**Таблица rental (записи о взятии и возврате кассет)**
| Поле           | Тип     | Описание                            |
|----------------|---------|-------------------------------------|
| id             | INTEGER | Идентификатор операции              |
| user_id        | INTEGER | Идентификатор пользователя из users |
| tape_id        | INTEGER | Идентификатор кассеты из tapes      |
| borrowing_date | TEXT    | Дата взятия кассеты на прокат       |
| return_date    | TEXT    | Дата возвращения кассеты            |
